## Guide Setup

Make sure to install the dependencies:

```bash
# env
cp .env.example .env
# make sure database pointing your local database server

# install package
composer install

# generate key
php artisan key:generate

# run migration 
php artisan migrate

# run seeder
php artisan db:seed

# run testing
php artisan test

#run app locally 
php artisan serve

```
